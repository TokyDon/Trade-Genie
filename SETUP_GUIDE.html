Key in .env file Required? Where to get it
OPENAI_API_KEY ✅ At least one LLM platform.openai.com → API Keys

GMAIL_ADDRESS For email reports Your Gmail address e.g. you@gmail.com
GMAIL_APP_PASSWORD For email reports myaccount.google.com/apppasswords — create one for "Mail".

Requires 2FA to be enabled on your Google account first.

⚠️ Never share your .env file — it contains your private API keys. It is excluded from GitHub automatically.
7
Check your configuration

You'll see ✅ for each service configured correctly. You need at least one LLM showing ✅ to run analysis.

8
Run your first manual analysis (test run)

Takes 2–5 minutes. Collects data from all sources, runs 3 LLMs in parallel, cross-checks results, and prints your top picks. When done, press Ctrl + C to stop before the next step.

✅ If you see picks printed with confidence scores, everything is working.
